## Evidence Checklist

Attach all of the following before a release can be approved:

- Screenshot of app launch and tray presence.
- Screenshot or short screen recording of `Ctrl+K` opening Quick Connect.
- Screenshot or screen recording of one SSH session opening successfully.
- Screenshot or screen recording of one serial session opening successfully, if serial is in scope.
- Screenshot or short screen recording showing broadcast mode warning banner, if broadcast is enabled.
- Any failing commands, logs, or notes captured during the run.
